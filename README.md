# Coronator

Surveillance app to do a perfect contact tracing and prevent more coronavirus infection

## How to Contribute

### Prerequisites

1. Flutter
2. [Core-onator](https://github.com/coronatorid/core-onator/)

### Installation

1. Copy env.sample
2. Run the application `make run`
